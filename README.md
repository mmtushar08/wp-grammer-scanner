## Description
WP Grammar Scanner automatically checks all WordPress content for grammar and spelling mistakes using the free LanguageTool API.  
It provides suggestions in the admin dashboard and allows you to schedule scans monthly or manually.

## Features
- Scan all posts, pages, and custom post types
- Show grammar & spelling errors
- Suggest corrections (do not auto-apply)
- React-based admin dashboard
- Reports with DataTable
- PDF export & email summary
- Cron scheduling for automated scans

## Installation
1. Upload the plugin folder `wp-grammar-scanner` to `/wp-content/plugins/`
2. Activate the plugin via WordPress admin
3. Navigate to **WP Grammar Scanner** in admin menu
4. Configure settings & start scanning
